To Use:
-npm init -y

-Install Dependencies:
1) Typescript (programming language to use)
	*npm install typescript --save-dev
	*npx tsc --init (for compiling)
2) Axios (promise based HTTP client for the browser and node.js)
	*npm install axios --save-dev
3) Jest (JavaScript testing framework)
	*npm install jest --save-dev 
	*npx ts-jest config:init (for Jest configuration file)
	*npm i -D ts-jest @types/jest  (preprocessor; to test projects written in TypeScript)
